summary: Turn on 'avoid ever using icon title' by default
class: wish: This is a request for an enhancement.
difficulty: fun: Just needs tuits, and not many of them.
priority: medium: This should be fixed one day.
fixed-in: 2003-01-23 dbee2d886fa418c82ac9e6a121ff6b29508bcf07 2003-01-24 (0.54)

Separate window and icon titles don't make as much sense in the post-Win95 world as in X11/Windows 3 style user interfaces. Judging by the mail we get, the resulting behaviour seems to be surprising to a fair few users. Perhaps we should turn off use of icon titles by default.

In fixing this, I've changed the sense of the control in the Windows UI - what was `Avoid ever using icon title' is now `Separate window and icon titles'. Docs updated accordingly.

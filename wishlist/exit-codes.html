summary: Provide more useful exit codes
class: bug: This is clearly an actual problem we want fixed.
depends: psftp-batch-exit-code

A common complaint is that the exit codes of various PuTTY tools on various platforms aren't as useful as they might be.

We should review, fix, and document the exit codes of the various PuTTY tools (ensuring we disambiguate local status from the exit code provided by an SSH server, where applicable).

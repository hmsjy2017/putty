summary: Auto-scrolling while selecting should have variable speed
class: wish: This is a request for an enhancement.
difficulty: fun: Just needs tuits, and not many of them.
priority: medium: This should be fixed one day.

A present, if term_mouse() gets a drag event above or below the
window, it scrolls the window by one line.  It would be better if
the amount of scrolling were proportional to the distance by which
the mouse pointer was outside the window.
Audit trail for this wish.

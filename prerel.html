This page contains download links for a pre-release version of PuTTY 0.75.

Pre-release versions of PuTTY are likely to be substantially the same as the next release, when we release it: we expect to only fix bugs, and not add any new features that will impact the stability of the code.

The development snapshots may contain features that are not in this pre-release and will not be in the 0.75 release either.

If you find a bug in this pre-release, then please report it, in the hope that we can fix it before the actual 0.75 release!
